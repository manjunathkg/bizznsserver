Bizzns Projects

This is the root folder for all bizzns bundles that can be deployed to a 
osgi server like apache sling.

DEVELOPMENT
0) Install java 1.6.* and set JAVA_HOME/bin in PATH
1) Need Eclipse latest - this is STS 2.9.1
2) Install maven 3.0.3 or greater + mvn eclipse plugin. Set MAVEN_HOME
3) Install git and github client.




INSTALLATION 

1) Make sure your Apache Sling is started or know amazon instance where it is running
2) Launch the next maven command on projects. 
 
3) Browse your project at http://{amazon aws machinename}:8080/{project_path}.html
